Sound Studio

Every sound here is generated live by the hayao engine from a few dozen lines of data — no audio files, no samples. The music is deterministic and machine-verified: the same Song renders byte-identical PCM in Node, so it's tested like any other code.

synthesis: pure TypeScript · zero dependencies · rendered headlessly & feature-tested · played through the Web Audio API

Music — five genres, composed as data

Click to play a looping track. Each is built with the engine's theory helpers (scales, diatonic chords, roman-numeral progressions, voice-leading) and shows its rendered waveform + spectrogram.

▶ play a track to see its waveform & spectrogram

Sound effects — parametric, one line each

Each SFX is a plain SoundSpec object synthesized on the fly (ZzFX-style, but named & deterministic).

Built with hayao · the audio engine is documented in docs/AUDIO.md. This whole page drives the real published API — renderSong, audio.playSong, audio.playSpec, renderAudioFilmstrip.
